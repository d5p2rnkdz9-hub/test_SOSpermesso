# Requirements: Corso AI Pre-Training Screening

**Defined:** 2025-02-04
**Core Value:** Understand what each participant knows about AI before the course starts

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Assessment Core

- [ ] **ASMNT-01**: User can answer multiple choice questions (select one)
- [ ] **ASMNT-02**: User can answer multiple select questions (select all that apply)
- [ ] **ASMNT-03**: User can answer true/false questions
- [ ] **ASMNT-04**: Quiz flow branches based on user answers (adaptive path)
- [ ] **ASMNT-05**: User session persists across page refresh/close (can resume)

### Results & Feedback

- [ ] **RSLT-01**: User receives personalized feedback based on their specific answers
- [ ] **RSLT-02**: User receives recommendations for what to focus on before course

### Admin Dashboard

- [ ] **ADMN-01**: Admin can view list of all participant responses
- [ ] **ADMN-02**: Admin can view individual participant's full response details
- [ ] **ADMN-03**: Admin can export all responses as CSV
- [ ] **ADMN-04**: Admin can see analytics (question performance, completion rates)

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Rules & Scoring

- **RULES-01**: System evaluates answers using custom rules engine
- **RULES-02**: System automatically assigns proficiency level (1, 2, or 3)
- **RULES-03**: Rules can consider specific answer combinations

### Results Enhancement

- **RSLT-03**: User sees their assigned level immediately after completion
- **RSLT-04**: User sees explanation of their level assignment

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Video content | Not needed for screening tool |
| Payment processing | Handled separately from assessment |
| LMS integration | Standalone for v1, integrate later |
| Mobile app | Web-first, responsive design sufficient |
| Multi-language | Italian only for now |
| User accounts/login | Not needed if using enrollment links |
| Question bank management | Admin creates questions directly in code/DB for v1 |
